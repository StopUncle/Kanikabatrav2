## Usage Notes

### For Dialogue Writing
1. Read the character's voice patterns in their file
2. Check their "Never" rules
3. Use their tells/behavioral cues
4. Match their default emotion + variations

### For Scene Building
1. Check character's locations list
2. Verify path alignment
3. Use appropriate emotion for context
4. Reference their arc position

### For Inner Voice
- ONLY in choice scenes
- ALWAYS as last dialog item
- Describe, don't explain
- Short and punchy
