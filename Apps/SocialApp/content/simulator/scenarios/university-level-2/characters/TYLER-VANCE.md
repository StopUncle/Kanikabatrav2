## Clinical Profile

### DSM-5 Markers (HPD)
- Must be center of attention (core need)
- Dramatic, theatrical, exaggerated emotions
- Uses appearance to draw attention
- Emotions shift rapidly
- Suggestible, easily influenced
- Shallow emotional expression
- Self-esteem dependent on others' approval

### Core Wound
**Invisibility anxiety** - Tyler cannot tolerate being ignored. Any moment without attention feels like dying.

## Dialogue Analysis

### Voice Patterns

#### Default Mode (Performance)
```
"Oh my GOD, you won't BELIEVE what happened to me."
"Everyone was staring. It was so embarrassing." [loved it]
"This one time..." [launches into long story]
```
- Everything is dramatic
- Every story features him prominently
- Exaggerated emotional delivery
- Needs constant audience reaction

#### Attention-Seeking Mode
```
"Anyway, as I was SAYING..."
"Hello? Did anyone hear me?"
"Fine. I guess no one cares."
"Did you see what I did there? Pretty good, right?"
```
- Escalates when ignored
- Fishing for compliments
- Creates drama if attention wanes
- Cannot tolerate being overlooked

#### Useful Mode (Trading Information)
```
"You want to know who's really running things here? Let me tell you..."
"I know everyone. And everyone's secrets."
"You need to get into VIP? I can make that happen. But first..."
```
- Likes feeling important
- Trades access for attention/validation
- Actually has valuable intel
- Connects people (for the credit)

#### Wounded Mode (Ignored)
```
"Whatever. I'm bored."
"I have other places to be."
"Call me when you're ready to have fun."
```
- Retreats when can't reclaim spotlight
- Sulks dramatically
- May become hostile to attention-stealer
